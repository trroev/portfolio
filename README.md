## How It's Made:

**Built with:**

- [React](https://reactjs.org/)

- [Lucide](https://lucide.dev/)

## Getting Started:

```
# clone this repo
git clone https://github.com/trroev/portfolio.git
# change into the directory
cd portfolio
# install dependencies
npm install
# start development server
npm start
```
